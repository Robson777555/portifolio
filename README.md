## 🚀 Como Usar

### Visualização Local
1. Baixe todos os arquivos do portfólio
2. Abra o arquivo `index.html` em um navegador web
3. Navegue pelas seções usando o menu ou scroll

### Hospedagem
Para hospedar o portfólio online:
1. Faça upload de todos os arquivos para seu servidor web
2. Certifique-se de que o arquivo `index.html` está na raiz
3. Configure o servidor para servir arquivos estáticos

### Personalização
Para personalizar o portfólio:
1. **Conteúdo**: Edite o arquivo `index.html`
2. **Estilos**: Modifique o arquivo `css/style.css`
3. **Interatividade**: Ajuste o arquivo `js/main.js`
4. **Cores**: Altere as variáveis CSS no início do arquivo de estilos
